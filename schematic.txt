ПИНОВКА ARDUINO NANO:
────────────────────
           ┌───┐
       D13 │○ ○│ D12
       D11 │○ ○│ D11
       D10 │○ ○│ D10  → SERVO 2
        D9 │○ ○│ D9   → SERVO 1
        D8 │○ ○│ D8
        D7 │○ ○│ D7
        D6 │○ ○│ D6
        D5 │○ ○│ D5
        D4 │○ ○│ D4
        D3 │○ ○│ D3
        D2 │○ ○│ D2   → PWM INPUT
       GND │○ ○│ D1
      RESET│○ ○│ D0
        RX │○ ○│ RX1
        TX │○ ○│ TX1
        5V │○ ○│ 5V   → Питание для PWM источника
       GND │○ ○│ GND
       VIN │○ ○│ VIN  → Внешнее питание 7-12V
           └───┘

ЦВЕТОВАЯ МАРКИРОВКА ПРОВОДОВ:
─────────────────────────────
• КРАСНЫЙ   → +5V (питание)
• ЧЁРНЫЙ    → GND (земля)
• ЖЁЛТЫЙ    → SIGNAL (управление)
• ОРАНЖЕВЫЙ → PWM INPUT (вход)
• СИНИЙ     → Внешнее питание

РЕКОМЕНДУЕМЫЕ КОМПОНЕНТЫ:
────────────────────────
• Конденсаторы 100µF на линиях питания серв
• Диод 1N4007 для защиты от обратной полярности
• Стабилизатор напряжения 5V 3A для серв
• Разъемы JST для подключения серв

ПРИМЕЧАНИЯ:
──────────
1. Не подключайте сервы напрямую к 5V пину Arduino
2. Используйте отдельный источник питания для серв
3. Все GND должны быть соединены (общая земля)
4. Проверьте полярность подключения
5. Используйте конденсаторы для фильтрации питания


🔌 Схема подключения для 10В питания:

┌─────────────────────────────────────────────┐
│              10V ПИТАНИЕ                    │
│  (стабилизированный источник 10В, 5А+)     │
└───────────────────┬─────────────────────────┘
                    │
┌───────────────────▼─────────────────────────┐
│             ARDUINO NANO                    │
│                                             │
│  D2 ───────► PWM сигнал с ресивера         │
│  D3 ───────► Светодиод (220Ω)              │
│  D5 ───────► Драйвер IN1 (левый магнит)    │
│  D9 ───────► Драйвер IN3 (правый магнит)   │
│  GND ──────► Общая земля                   │
│  5V ───────► Питание 5V (от 10В через LDO) │
└───────────────────┬─────────────────────────┘
                    │
┌───────────────────▼─────────────────────────┐
│          ДРАЙВЕР MX1508 / ТРАНЗИСТОРЫ       │
│                                             │
│  IN1 (D5) ──────► Управление левым         │
│  IN3 (D9) ──────► Управление правым        │
│  GND ───────────► Земля                     │
│  VCC ───────────► 10V питания              │
│  OUT1 ──────────► Левый магнит (+)         │
│  OUT3 ──────────► Правый магнит (+)        │
│  Магниты (-) ───► Земля                    │
└─────────────────────────────────────────────┘

ВАЖНО: Для 10В питания Arduino нужен стабилизатор 5В (LDO)
или отдельный источник 5В для Arduino.
